## 项目下载
```
git clone https://gitee.com/Jmysy/pear-admin-ant.git
```

## 项目跳转
```
cd pear-admin-ant
```

## 项目安装
```
npm install
```

### 项目运行
```
npm run serve
```

### 编译项目
```
npm run build
```
